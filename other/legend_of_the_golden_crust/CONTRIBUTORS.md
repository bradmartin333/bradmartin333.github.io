# Contributors

## Legend of the Golden Crust

This game is a collaborative effort to create an entertaining browser-based arcade experience.

### Core Game Design
- **Original Concept**: bradmartin333
- **Game Design**: bradmartin333

### Development & Features

#### Major Feature Contributions
- **Code Refactoring & Organization**: Modular architecture with separate JS modules (config, sprites, utils, game)
- **Collision Detection Improvements**: More forgiving collision system with padding for better gameplay
- **Temperature/Warmth System**: Added warmth mechanic with space heaters, sterno fuel, and burning couches
- **Weapon Upgrades**: Fleeting power-ups including multishot, speed shot, and flame shot
- **Enemy Drop System**: Bad items that negatively affect player (slow, damage, cold, horde spawns)
- **MEGA MODE**: Special mode triggered by high health, high warmth, and killing spree
- **Spawn System Fixes**: Improved spawn logic to prevent items/enemies spawning inside obstacles
- **UI Enhancements**: Added warmth bar, updated controls menu, and last updated footer

### How to Contribute
If you'd like to contribute to this project:
1. Fork the repository
2. Create a feature branch
3. Make your changes with clear, commented code
4. Test thoroughly
5. Submit a pull request with a detailed description

### Reporting Issues
Please report bugs or suggest features by opening an issue in the repository.
